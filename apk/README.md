# Lingua Mini — Android APK

Файл `lingua-mini.apk` — готовая установочная сборка приложения для Android.

## Как установить
1. Скачай `lingua-mini.apk` на телефон (кнопка **Download** / «Скачать» на странице файла в GitHub).
2. Открой файл на телефоне.
3. Если появится предупреждение — разреши установку из неизвестных источников для браузера/файлового менеджера.
4. Установи и запусти **Lingua Mini**.

> Это debug-сборка для тестирования (не из Google Play), подписана debug-ключом.
> Свежий APK также автоматически собирается в GitHub Actions и публикуется в разделе **Releases** (тег `android-latest`).

## Что внутри
- Карточки (140 слов, 13 категорий) с возможностью **скрывать** известные слова.
- Квизы по словам.
- **Грамматика**: времена (теория + практика), неправильные глаголы, слова-исключения.
- Прогресс, серии (streak), статистика.
